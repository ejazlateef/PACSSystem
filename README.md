# PACS System

This project is a Picture Archiving and Communication System (PACS) designed for managing medical imaging. It uses EJS for dynamic content rendering and allows users to interact with images and data based on their roles: Admin, Radiologist, and Reviewer.

## Features

- **Admin Role**: Admins can manage users, set document requirements, and generate reports.
- **Radiologist Role**: Radiologists can view, upload, and manage medical images (DICOM), and create reports for patients.
- **Reviewer Role**: Reviewers can evaluate images, provide feedback, and manage reviewer processes.
